#### Requirements

Install Python3, then:

```
python3 -m venv novanav-env
source novanav-env/bin/activate
```

```
pip3 install -r requirements.txt
```

#### Runt it

```
python3 ./src/novanav.py
```

#### How-to

```
ctrl+t (New tab)
ctrl+v (hide tabs for distraction-free)
ctrl+q (quit)
```
